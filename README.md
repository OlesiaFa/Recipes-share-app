# Recipes-share-app
> The Recipes app is a web application built on the MERN stack. It provides users with the ability to create accounts, post recipes, view recipes, shared by others, seasrch by name and save their favorite recipes.

## Technologies Used

- Frontend: React.js
- Backend: Node.js, Express.js, MongoDB, JSON Web Tokens (JWT)
  
## Features

- Users can register and login securely to access the app.
- Users can create, edit, and delete their own recipes.
- Users can view recipes posted by other users.
- Users can save recipes posted by others to their profile.

## Setup
- Clone the repository:
  `git clone https://github.com/OlesiaFa/Recipes-share-app/`

- Install dependencies:
   `npm install`
- Start terminal:
    `npm run dev`

## Usage

-  Register a new account or login with existing credentials.
-  Navigate to the recipes section to view and post recipes.
-  Save recipes from other users to your profile for quick access.
